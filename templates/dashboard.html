📝 Ringkasan Penelitian

Penelitian ini mengevaluasi kinerja metode Support Vector Machine (SVM) dalam mengklasifikasikan penduduk miskin yang layak menerima bantuan sosial berdasarkan indikator sosial ekonomi.

Untuk meningkatkan akurasi dan menyederhanakan model, Algoritma Genetika (GA) digunakan sebagai metode pembungkus (wrapper) seleksi fitur. GA menyeleksi subset fitur optimal yang memiliki tingkat kontribusi tinggi, mengeliminasi fitur-fitur yang tidak relevan (noise), dan mempercepat waktu komputasi pelatihan SVM.

Dataset: Berisi informasi kependudukan terperinci termasuk identitas penduduk (NIK & Nama) serta parameter numerik dan kategorik penunjang.

🗺️ Alur Pipeline GA-SVM

1

Memuat Dataset

Mengunggah file CSV atau memuat dataset sampel bawaan dengan NIK & Nama penduduk.

2

Preprocessing Data

Menghapus data duplikat, mengisi missing value, meng-encode variabel kategori, dan memisahkan kolom identitas.

3

Split Train-Test

Membagi data latih (80%) dan data uji (20%) secara stratified untuk menjaga distribusi kelas target.

4

SVM Baseline & Seleksi Fitur GA

Melatih model SVM baseline, kemudian menjalankan Algoritma Genetika untuk memilah kombinasi fitur terbaik.

5

Evaluasi Akhir

Membandingkan akurasi, presisi, recall, F1-Score, dan confusion matrix sebelum vs setelah optimasi.
